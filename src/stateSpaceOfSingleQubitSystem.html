The State Space of a Single-Qubit System

The state space of a classical or quantum physical system is the set of all possible states of the system.

Depending on which properties of the system, a state consists of any combination of the positions, momenta, polarizations, spins, energy, and so on.
When we are considering only polarization states of a single photon, the state space is all possible polarizations.
More generally, the state space for a single qubit, no matter how it is realized, is the set of possible qubit values

Global phase

For us unit vectors that differ only for a phase are equivalent and represent the same state. $$ | v' \rangle= c| v \rangle \\ c= e^{i\phi} $$ The multiple by which two vectors representing the same quantum state differ is called the global phase and has no physical meaning. $$ | v' \rangle \sim c| v \rangle $$

Relative Phases and equivalent state vector

$$ |v \rangle = a|0 \rangle + b|1\rangle $$ The relative phase (in the standard basis) is the phase difference in between the two complex numbers a and b.
Two superpositions of two state vector whose amplitudes have the same magnitudes but that differ in a relative phase represent different states.
Relative phase is meaningful.
Global phase has no physical meaning.
